Catálogo GIMP para Glade

11 de outubro de 2011

Estou planejando desenvolver alguns plugins para o GIMP, e eu gostaria de projetar as interfaces gráficas usando o Glade. Infelizmente não há um catálogo dos widgets do GIMP para o Glade, então, mais um projeto que entra na pilha.

Desde a versão 3.10.0 o Glade está gerando arquivos somente para o GTK+3, entretanto projeto GIMP está planejando compatibilidade com o GTK+3 somente a partir da versão 3.10 (a última versão estável é a 2.6.11). Portanto será necessário usar o Glade 3.8.0 (ou menor) para gerar um arquivo .ui compatível com o GTK+2.

O catálogo consiste de um arquivo XML e alguns ícones, sendo que eu ainda não fiz os ícones, principalmente por causa da minha falta habilidade com desenhos. Para ser usável, o arquivo XML deve ser instalado no diretório do sistema de catálogos do Glade (normalmente /usr/share/glade3/catalogs) ou em um diretório listado na variável de ambiente GLADE_CATALOG_PATH.

O projeto está hospedado no GitHub (aqui).
